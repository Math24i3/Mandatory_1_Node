Tools for Node:

This section will cover the setup in order to get a simple node application/listener up and running.

Creating a Node.Js project:

- In order to setup a simple node project you first need to create a project folder.

- Next up is to create a package.json file.

The package.json file contains information about the project, dependencies, settings etc.

NPM:

NPM stands for Node Package Manager. It manages all the packages and modules you have installed or wish to install/implement to your project.

Running NPM commands:

- After setting up the project structure (almost) it's time to run the command: npm install express or npm install.

The command(s) install the dependencies and the necessary files to the project.

If you type the module name after the command npm install (ex. express), you specify the module you wish to implement. Otherwise it will install all the dependencies you have specified in the package.json file.

Get your Node application up'n running:

- Next up is to create the application-runnable file called app.js.

The app.js file contains the routes and functions that are required in order to run the application.

The main purpose of app.js is to serve http requests from clients.
